# LKID-35: Build 7 Prototype Screens with Navigation

## Delivered
- [x] Landing page (/) with hero CTA
- [x] Auth page (/auth) with email entry, magic-link-sent, and expired views
- [x] Predict page (/predict) with lab value form (BUN, creatinine, potassium, age)
- [x] Results page (/results) with eGFR trajectory chart and PDF download
- [x] Shared Header component with navigation
- [x] DisclaimerBlock component (mobile sticky + desktop footer)
- [x] Responsive layout: mobile single-column, tablet/desktop 2-column grid
- [x] Input validation attributes (min/max/step) on numeric fields
- [x] Accessibility: aria-expanded, aria-controls, aria-required attributes

## Run locally
```bash
cd app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build
```bash
cd app
npm run build
```
