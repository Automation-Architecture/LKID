# Inga -- UX/UI Drafts

Discovery Phase, Step 5: Parallel Artifact Drafting.

## Deliverables

| File | Description |
|------|-------------|
| `user-flows.md` | Complete user flow diagrams for all paths: landing, form, results, save prompt, account creation, multi-visit re-entry, error recovery, session lifecycle |
| `wireframes.md` | Text-based wireframes for all screens at 3 breakpoints (mobile <768px, tablet 768-1024px, desktop >1024px) |
| `component-specs.md` | Detailed component specifications: props interfaces, visual states, variants, spacing, responsive behavior, accessibility per component |
| `design-tokens.md` | Finalized design tokens mapped to Tailwind CSS variables with globals.css and tailwind.config.ts references |
| `chart-specs.md` | Two chart interaction specification variants (Variant A: full Visx interactive, Variant B: simplified static fallback) |
| `accessibility-plan.md` | WCAG 2.1 AA compliance checklist, chart-specific accessibility, keyboard navigation map, screen reader flow, testing plan |
